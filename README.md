An intelligent console-based healthcare assistant built using Python that performs symptom analysis, risk assessment, emergency detection, specialist recommendation, and OTC medicine suggestions using Artificial Intelligence concepts such as Knowledge Representation, Rule-Based Inference, Decision Making, and Expert Systems.
CO1 – Intelligent Agents, Problem Formulation & Knowledge Representation
CO2 – Search and Problem Solving
CO3 – Constraint Satisfaction and Explainable Reasoning
CO4 – Decision Making and Utility-Based Reasoning
CO5 – Probabilistic Reasoning and Uncertainty Handling
CO6 – Hybrid AI System Integration
The project demonstrates the practical application of Artificial Intelligence concepts including Intelligent Agents, Knowledge Representation, Expert Systems, Inference Engines, Decision Making, Probabilistic Reasoning, Explainable AI, and Hybrid AI Architectures within the domain of Telemedicine and Healthcare Assistance.
